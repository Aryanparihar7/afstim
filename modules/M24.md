# M24 — Reflection UI + API

Track: Validation · Depends: M14, M04 · Blocks: M25 · Agent time: ~2h

## Goal
The learner writes one honest sentence about what just happened, before we call it done.

## Files you may touch
- `src/components/afstim/reflection-block.tsx`
- `src/app/api/v1/missions/[slug]/reflection/route.ts`
- `src/features/progress/services/reflection-service.ts`

## Spec

**UI** — appears **only when all checks pass**. Before that, don't show it; a reflection box next to failing checks is noise.

```
eyebrow  BEFORE YOU MOVE ON
         Your app is now running somewhere that isn't your laptop.
         What has to be true about your code for that to work?
textarea (min 40 chars)
[Save and finish mission]
```

Questions come from the mission's `reflection` frontmatter array.

**API** — `POST /missions/[slug]/reflection`, session auth.
- Zod: 40–2000 chars
- Write `MissionProgress.reflection`
- Call `verification-service.evaluate()` → if verified, respond with the next mission slug
- `track('reflection_saved')`

**No AI grading.** Not now, not later in v0.1. Grading costs money, produces false negatives, and false negatives on a *reflection* — a thing with no right answer — would be the single most infuriating experience in the product.

Someone will type "asdf". Let them. They paid to learn; typing "asdf" only cheats themselves. The 40-char floor is the entire enforcement mechanism and it is enough.

**Copy:** never "correct" or "incorrect". Never a score. Save and move on.

## Out of scope
AI grading. Sharing reflections. Reading them back — **you** read them in Prisma Studio; that's your best signal about whether missions are landing.

## Done when
- [ ] tsc, lint, build pass
- [ ] Under 40 chars → inline message, no submit
- [ ] Reflection block hidden while any check fails
- [ ] Saving a valid reflection with all checks passing → mission verifies

## Founder gate
1. Pass all checks → the reflection question appears. ☐
2. Type "hi" → *"A bit more than that."* ☐
3. Write a real answer → mission verifies, next mission unlocks. ☐
4. **Read your own reflection back a week later.** If it's useless, the question is wrong — that's content, fix it in M35. ☐
