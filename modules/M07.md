# M07 — Feedback form + API

Track: Marketing · Depends: M02, M04 · Blocks: M34 · Agent time: ~2h

## Goal
Anyone can tell you something's wrong, in under 20 seconds, without an account.

## Files you may touch
- `src/app/(marketing)/feedback/page.tsx`
- `src/app/api/v1/feedback/route.ts`
- `src/features/feedback/**`

## Spec

**Page** — one card:
- Category: `Something's broken` / `An idea` / `This confused me` / `Something else`
- Message — textarea, required, 10–2000 chars
- Email — optional. Label: *"Only if you want a reply."*
- `[Send feedback]` → success state replaces the form: *"Sent. Thank you — this is read by one person and that person is me."*

**API** — `POST /api/v1/feedback`, public.
- Zod: `category` enum `bug|idea|confusing|other`, `message` 10–2000, `email` optional valid email
- Insert to `Feedback`. If a session exists, attach `userId`.
- Rate limit: 3 per IP per hour. Simple in-memory Map keyed by IP is **fine** at this scale — do not add Redis.
- Never echo the message back in the response.

**Public feedback is a spam target.** Cap length, cap rate, don't render it anywhere public. That's enough.

## Out of scope
Email notification to you (check `/admin/feedback`, M34). File uploads. Screenshots. A widget on every page. Categorisation logic.

## Done when
- [ ] tsc, lint, build pass
- [ ] 4th submission in an hour → clean 429, human message
- [ ] Empty message → inline error naming the field
