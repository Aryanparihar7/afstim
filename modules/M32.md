## Spec

**`/admin/ai`** — plain table. Admin only.

**Today**
```
Calls today          47

Estimated cost      ₹ 12.40

```

**Per user, last 7 days** — email · calls · tokens in · tokens out · est. ₹. Sorted by cost desc.

**Daily, last 30** — date · calls · est. ₹.

**`pricing.ts`**
```ts
export const HAIKU_INPUT_PER_MTOK_USD = 0    // ← FILL FROM anthropic.com/pricing
export const HAIKU_OUTPUT_PER_MTOK_USD = 0   // ← FILL FROM anthropic.com/pricing
```

**Agent: do not guess these numbers.** Leave them 0 and tell the founder to fill them from the pricing page. A hallucinated price here is worse than no price — it would give a false sense of safety about the one budget that can kill this.

Show `⚠ Pricing not configured` while they're 0.

**Alert** — if today's estimate exceeds ₹100, a banner: *"Today is running hot. Check the top user."*
