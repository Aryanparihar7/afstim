# M19 — Local check runners

Track: Validation · Depends: M18 · Blocks: M20 · Agent time: ~4h

## Goal
`npx afstim check` tells a learner the truth about their own machine.

## Files you may touch
- `cli/src/commands/check.ts`
- `cli/src/runners/command-succeeds.ts`
- `cli/src/runners/file-exists.ts`
- `cli/src/runners/file-contains.ts`
- `cli/src/runners/http-ok.ts`
- `cli/src/runners/index.ts`
- `cli/src/lib/ui.ts`

## Spec

**Four runners. Four. Not five.** If a mission needs a fifth, that's a founder decision, not an agent decision.

| Type | Behaviour |
|---|---|
| `command_succeeds` | `execa(run, { shell: true, cwd, timeout: 20_000, reject: false })`. Pass = exit 0 (+ optional `expectStdoutMatches` regex on stdout). |
| `file_exists` | Glob `path` from cwd. Pass = ≥1 match. Never traverse above cwd. |
| `file_contains` | Read `path`, test `pattern` regex. Missing file = fail with *"That file doesn't exist yet."* |
| `http_ok` | `fetch(url)`, 10s timeout. Pass = `expectStatus`. Optional `expectJsonPath` → `{ path, equals }`. |

**Run in order, print as you go.** Never batch and dump — a learner watching rows resolve is the entire feeling of this product.

**Failure output** is the mission's `hint` and nothing else. Never dump stderr into the terminal — the learner already has it in their own terminal.

**Timeouts are hard.** A hung `npm run build` must not hang the CLI forever. 20s, then fail with *"That took longer than 20 seconds. Something's stuck."*

**Guards:**
- Refuse to run outside a directory with a `package.json` or `.git` → *"Run this from your project folder."*
- Never write, create, move, or delete anything in the learner's project. **Read and run only.**
- Never `eval`. Never run a string that didn't come from the mission file's `run` field.

**Exit code:** 0 if all pass, 1 if any fail. Scripts and CI depend on this.

## Out of scope
Posting results (M20). Server checks (M21). Retry loops. Watch mode.

## Done when
- [ ] `npm run build` in `cli/`
- [ ] All four types work against a real scratch project
- [ ] A command that sleeps 60s times out at 20s
- [ ] Running in `/tmp` (no package.json) refuses with a plain sentence

## Founder gate
1. Scratch folder → `afstim check` → real pass/fail against mission 1. ☐
2. Deliberately break something → the failing row shows **the hint you wrote**, not a stack trace. ☐
3. Watch the rows appear one at a time. **Does it feel good?** If it dumps all at once, reject. ☐
4. Run it in your Downloads folder → refuses politely. ☐
