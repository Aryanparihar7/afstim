Track: AI · Depends: M15, M22 · Blocks: M29 · Agent time: ~3h

Assemble exactly what the mentor needs to know, and nothing else.

## Spec

**Client** — `@anthropic-ai/sdk`. Model `claude-haiku-4-5`. `max_tokens: 700`. Temperature default. **No streaming in v0.1** — it's a help panel, not a chat.

**Context — hard cap 6,000 tokens. Assert it. Throw if exceeded.**

Include, in this order:
1. Mission slug, title, objective (frontmatter)
2. The failing check: `id`, `label`, `hint`
3. Learner's error text — **truncate to 2,000 chars**
4. Last 4 turns for this mission (from `AiMessage`)
5. Completed mission slugs — **names only**

**Never include:**
- File contents, repo contents, directory listings
- Environment variables, connection strings, tokens, secrets
- The learner's email, name, or ID
- Full stack traces beyond the 2,000 chars
- The whole mission MDX (objective only)

**System prompt** — `PLAN.md` Section 9, **verbatim**. Do not "improve" it. In particular do not add "guide them to discover the answer themselves." That instruction is the single most requested change and it is wrong: the learner has ChatGPT in the next tab. A mentor that withholds loses to alt-tab in five seconds and earns resentment on the way out. **We sell context, not restraint.**

## Out of scope
The API route (M29). UI (M30). Conversation memory across missions. Streaming. Model fallback. RAG. Embeddings. Fine-tuning. **A second model tier — Haiku is the whole strategy.**
