# M30 — Mentor panel UI

Track: AI · Depends: M29, M02 · Blocks: — · Agent time: ~3h

## Goal
Asking for help takes one click and no explaining.

## Files you may touch
- `src/components/afstim/mentor-panel.tsx`
- `src/app/(app)/mission/[slug]/page.tsx` (slot it in)

## Spec

**Placement** — inline on the mission page, **below the checks**. Not a docked right panel (there's no editor to sit beside). Not a floating bubble. Not a modal.

Your docs specify a permanently docked right-hand AI panel. That was designed for a three-panel IDE that no longer exists. In a reading page, a docked panel is a permanent distraction from reading.

**Collapsed** (default) — a single line under the failing checks:
```
Stuck on "Your app builds without errors"?   [Ask the mentor]
```
Appears **only when a check is failing.** No failure → no mentor. Nothing to be stuck on.

**Expanded:**
```
eyebrow  MENTOR · 22 LEFT TODAY

         About: Your app builds without errors

label    Paste the error you're seeing
textarea (mono, 6 rows)

label    What did you try?          ← optional
input

[Ask]
```

**Answer** — prose, `--text`, code in mono. Below it: `[Ask something else]`. Keep the last 4 turns visible, collapsed above.

**Not a chat app.** No bubbles. No avatars. No typing dots. No "Mentor is thinking...". No timestamps. Your Design System says it: *"The interface should never imitate a messaging application."*

**Loading** — the `[Ask]` button reads `Thinking…` and disables. That's all.

**Errors** — 429 → *"You've used today's questions. They reset at midnight."* 503 → *"The mentor is unavailable right now. Your work is unaffected."* Both inline, both calm, neither red.

**Prefill** — `checkId` and the check label come from the failing row. **The learner never explains what mission they're on.** That is the entire product advantage over the ChatGPT tab: it already knows.

## Out of scope
Streaming. Markdown rendering beyond code/paragraphs/lists. Copy button. Feedback thumbs. History page.

## Founder gate
1. Fail a check → the mentor line appears **under that check**. ☐
2. Click. **You didn't type what mission you're on. It already knew.** ☐
3. Paste a real error → useful answer. ☐
4. Does it look like WhatsApp? If yes, reject. ☐
5. Ask yourself: *is this better than the ChatGPT tab?* If the only difference is it's worse, the system prompt is wrong — go back to M28. ☐
