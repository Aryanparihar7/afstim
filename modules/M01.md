# M01 — Design tokens, fonts, globals

Track: Foundation · Depends: M00 · Blocks: M02 · Agent time: ~2h

## Goal
The monochrome design system exists as CSS variables and Tailwind theme values. No component can invent a colour after this.

## Files you may touch
- `src/app/globals.css`
- `tailwind.config.ts`
- `src/app/layout.tsx`
- `.eslintrc.json` (or `eslint.config.mjs`)

Anything else → **STOP AND ASK**.

## Spec

**1. Fonts** via `next/font/google` in `layout.tsx`:
- `Instrument_Sans` → CSS var `--font-sans`
- `JetBrains_Mono` → CSS var `--font-mono`

Do **not** use Inter or Geist. Both are the default look for developer tools; Geist reads as a Vercel clone.

**2. Tokens** in `globals.css` under `:root` — exactly these, no additions:

```css
--bg: #0A0A0B;
--surface: #111113;
--surface-raised: #17171A;
--border: #202024;
--border-strong: #2C2C32;
--text: #EFEFF1;
--text-muted: #8B8B93;
--text-subtle: #5A5A62;
--action-bg: #EFEFF1;
--action-fg: #0A0A0B;
--action-hover: #FFFFFF;
--pass: #7FB08A;
--fail: #C97B7B;
--pending: #5A5A62;
--r-sm: 6px; --r-md: 10px; --r-lg: 14px;
--t-fast: 120ms; --t-base: 200ms;
--ease: cubic-bezier(0.16, 1, 0.3, 1);
```

**3. Tailwind theme** — map every token to a semantic name (`bg-bg`, `text-muted`, `border-border`, `bg-action`). Delete Tailwind's default colour palette so `bg-blue-500` cannot be typed by accident.

**4. Spacing** — restrict the scale to: 1, 2, 3, 4, 6, 8, 12, 16, 24, 32 (×4px).

**5. `<body>`** — `bg-bg text-text font-sans antialiased`. Dark only. No theme switcher.

**6. Reduced motion:**
```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; }
}
```

**7. ESLint rule** banning raw hex in `src/components/**` and `src/app/**` (except `globals.css`).

## Out of scope
Light theme. Theme switching. Any component. Any page. Any accent hue — **there is no brand colour; white is the accent.**

## Done when
- [ ] `npx tsc --noEmit && npm run lint && npm run build` all pass
- [ ] `bg-blue-500` in a component fails lint or does nothing

## Founder gate
1. Run `npm run dev`, open the site. Background is near-black, text is off-white. ☐
2. Text is not Inter. (Instrument Sans has a distinctly rounder `a` and `g`.) ☐
3. macOS → System Settings → Accessibility → Reduce Motion on. Nothing animates. ☐

## If it breaks
`git revert HEAD` → back to M00.
