## Why this mission exists

Every mission after this one assumes three things are already true: you can run JavaScript locally, you can save your work with Git, and you can talk to GitHub from your terminal. Missing any one of them turns mission 3 or 4 into a tools problem instead of a shipping problem.

## What you're going to do

1. Install Node.js — the current LTS version — from nodejs.org, or with a version manager if you already use one.
2. Install Git from git-scm.com.
3. Install the GitHub CLI from cli.github.com.
4. Open a new terminal window. A terminal you already had open won't see tools you just installed.
5. Run each command below and confirm it prints a version number instead of an error.
